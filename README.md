Многие люди хотят включить медитацию в свой ежедневный ритуал, но сталкиваются с проблемой отслеживания времени. Приходится использовать сторонние приложения или часы. Основная задача - упростить этот процесс, предоставив удобный веб-инструмент прямо в браузере.

Цель проекта: разработать функциональный веб-сайт с таймером медитации, который позволит пользователям легко настраивать и контролировать длительность сессии.

Задачи проекта:
◦ создать минималистичный интерфейс с навигацией (три раздела: «Главная», «Таймер медитации», «Асаны»);
◦ реализовать таймер медитации на JavaScript с следующим функционалом:
     ✔ возможность установки длительности сессии (например, 10 минут);
     ✔ кнопка запуска и остановки отсчёта;
     ✔ визуальное отображение оставшегося времени (крупный циферблат в центре экрана);
     ✔ аудиосигнал при окончании времени отсчета.
◦ обеспечить плавную анимацию изменения времени;
◦ продумать эстетику: подобрать спокойные фоны, гармонирующие с темой медитации.

Технические детали реализации таймера:
◦ инициализация: получаем элементы для отображения времени (min_tens, min, sec_tens, sec);
◦ запус: пользователь задает время, запускается setInterval;
◦ расчет времени: на каждой итерации вычисляем оставшееся время (delta = end - Date.now()), преобразуем в минуты/секунды;
◦ обновление интерфейса: обновляем текст элементов с помощью #setTimer().
◦ завершение - при delta < 0:
      ✔ останавливаем таймер (clearInterval);
      ✔ воспроизводим аудиосигнал (audio.play());
      ✔ сбрасываем отображение времени.
◦ остановка вручную: метод #clearTimer() останавливает отсчёт и сбрасывает значения.

Пользователь заходит на сайт, выбирает длительность медитации (например, 10 минут), нажимает «Старт» - начинается отсчет времени с визуальным отображением на экране. По истечении времени раздаётся мягкий звуковой сигнал, уведомляющий о завершении сессии.

Целевая аудитория:
◎ люди, практикующие медитацию и ищущие простой онлайн-инструмент;
◎ занятые пользователи, которым важно экономить время на переключение между приложениями;
◎ новички в медитации, которым нужен чёткий временной ориентир для практики.
