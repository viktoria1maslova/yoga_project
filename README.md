Многие люди хотят включить медитацию в свой ежедневный ритуал, но сталкиваются с проблемой отслеживания времени. Приходится использовать сторонние приложения или часы. Основная задача - упростить этот процесс, предоставив удобный веб-инструмент прямо в браузере.<br>

<b>Цель проекта:</b> разработать функциональный веб-сайт с таймером медитации, который позволит пользователям легко настраивать и контролировать длительность сессии.<br>

<b>Задачи проекта:</b><br>
◦ создать минималистичный интерфейс с навигацией (три раздела: «Главная», «Таймер медитации», «Асаны»);<br>
◦ реализовать таймер медитации на JavaScript с следующим функционалом:<br>
     ✔ возможность установки длительности сессии (например, 10 минут);<br>
     ✔ кнопка запуска и остановки отсчёта;<br>
     ✔ визуальное отображение оставшегося времени (крупный циферблат в центре экрана);<br>
     ✔ аудиосигнал при окончании времени отсчета.<br>
◦ обеспечить плавную анимацию изменения времени;<br>
◦ продумать эстетику: подобрать спокойные фоны, гармонирующие с темой медитации.<br>

<b>Технические детали реализации таймера:</b><br>
◦ инициализация: получаем элементы для отображения времени (min_tens, min, sec_tens, sec);<br>
◦ запус: пользователь задает время, запускается setInterval;<br>
◦ расчет времени: на каждой итерации вычисляем оставшееся время (delta = end - Date.now()), преобразуем в минуты/секунды;<br>
◦ обновление интерфейса: обновляем текст элементов с помощью #setTimer().<br>
◦ завершение - при delta < 0:<br>
      ✔ останавливаем таймер (clearInterval);<br>
      ✔ воспроизводим аудиосигнал (audio.play());<br>
      ✔ сбрасываем отображение времени.<br>
◦ остановка вручную: метод #clearTimer() останавливает отсчёт и сбрасывает значения.<br>

Пользователь заходит на сайт, выбирает длительность медитации (например, 10 минут), нажимает «Старт» - начинается отсчет времени с визуальным отображением на экране. По истечении времени раздаётся мягкий звуковой сигнал, уведомляющий о завершении сессии.<br>

<b>Целевая аудитория:</b><br>
◎ люди, практикующие медитацию и ищущие простой онлайн-инструмент;<br>
◎ занятые пользователи, которым важно экономить время на переключение между приложениями;<br>
◎ новички в медитации, которым нужен чёткий временной ориентир для практики.<br>

<b>Используемые технологии:</b><br>

✔ CSS: для стилизации и адаптивного дизайна;<br>
✔ JavaScript: для логики таймера и аудиосигналов;<br>
